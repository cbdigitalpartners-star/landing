# Impeccable — Design Guidelines

## Project Snapshot

- **What:** Marketing landing site for **C&B Digital Partners** — a custom-software studio based in Santiago, Chile.
- **Stack:** Static HTML + hand-written CSS (no framework, no build step). Inter via Google Fonts. Deployed on Vercel.
- **Language:** Spanish (es).
- **Pages:** `index.html`, `privacidad.html`, `terminos.html`. Shared `styles.css`.
- **Brand mark:** Navy "C&B" wordmark with a circuit-trace motif; tagline "Tecnología · Estrategia · Resultados". Assets in `assets/` (`logo-clean.png` for header/footer, `favicon.ico`).

### Existing design tokens (`styles.css` `:root`)

| Token | Value | Use |
|-------|-------|-----|
| `--ink` | `#0b1020` | Primary text / dark surfaces (CTA band, buttons) |

| `--muted` | `#6b7280` | Secondary text |

Type: Inter 400–800. Headings tight tracking (`-0.02em`/`-0.03em`), `clamp()` fluid sizing.

## Design Context

### Users
Chilean businesses (decision-makers: founders, ops/marketing leads) evaluating whether to hire a studio to build custom software — messaging integrations, automation, web platforms, fintech, education tools. They arrive skeptical and time-poor; the job to be done is *"prove this studio is competent and safe to trust with my project, fast."* Emotional goal: **confidence** — they should feel the work is solid, secure, production-grade.

### Brand Personality
**Modern · trustworthy · technically precise.** Voice is direct and concrete (real products in production, official APIs, encrypted data) — never hype or buzzword salad. Sounds like senior engineers who ship, not a marketing agency.

### Aesthetic Direction
Clean, tech-forward, light-first with deep-navy anchors and a blue→teal accent. Generous whitespace, restrained palette, intentional hierarchy. Dark mode is in scope (support via `prefers-color-scheme`, mirror tokens). 

**Anti-reference — explicitly NOT generic AI-template:** avoid identical equal-weight card grids, emoji-as-icons, everything-centered layouts, and stock gradient soup. Each section should earn its layout. Replace placeholder emoji icons with real iconography over time.

### Design Principles
1. **Evidence over claims.** Lead with real products, real numbers, real screenshots — concrete proof beats adjectives.
2. **Restraint.** Two accents max (blue + teal); navy + paper carry the rest. More whitespace, fewer elements.
3. **Intentional hierarchy.** Vary layout/rhythm between sections; never ship a uniform card grid just because it's easy.
4. **Accessible by default (WCAG AA).** ≥4.5:1 text contrast, visible focus rings, full keyboard nav, honor `prefers-reduced-motion`. Verify both themes.
5. **Production-grade polish.** Tight typography, aligned spacing scale, no layout shift (size images), motion that's subtle and purposeful — the craft itself signals the studio's competence.
